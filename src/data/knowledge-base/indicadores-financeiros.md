Base de conhecimento sobre KPIs financeiros e valores de referência para PMEs.

## 1. Indicadores de Liquidez

Medem a capacidade da empresa de honrar seus compromissos de curto prazo.

### Liquidez Corrente (LC)

```
LC = Ativo Circulante / Passivo Circulante
```

| Faixa | Classificação | Interpretação |
|-------|---------------|---------------|
| < 1,00 | CRÍTICO | Empresa não consegue pagar dívidas de curto prazo |
| 1,00 - 1,50 | ATENÇÃO | Margem apertada, requer monitoramento |
| 1,50 - 2,00 | SAUDÁVEL | Boa capacidade de pagamento |
| > 2,00 | EXCELENTE | Alta liquidez (avaliar se capital está ocioso) |

**Valor de referência PMEs**: [AJUSTAR] 1,50 a 2,00

### Liquidez Seca (LS)

```
LS = (Ativo Circulante - Estoques) / Passivo Circulante
```

| Faixa | Classificação | Interpretação |
|-------|---------------|---------------|
| < 0,80 | CRÍTICO | Alta dependência de venda de estoques |
| 0,80 - 1,00 | ATENÇÃO | Precisa vender estoque para honrar dívidas |
| 1,00 - 1,50 | SAUDÁVEL | Boa liquidez sem depender de estoques |
| > 1,50 | EXCELENTE | Alta liquidez imediata |

**Valor de referência PMEs**: [AJUSTAR] 1,00 a 1,20

### Liquidez Geral (LG)

```
LG = (Ativo Circulante + Realizável LP) / (Passivo Circulante + Exigível LP)
```

| Faixa | Classificação | Interpretação |
|-------|---------------|---------------|
| < 0,80 | CRÍTICO | Patrimônio líquido negativo provável |
| 0,80 - 1,00 | ATENÇÃO | Endividamento alto |
| 1,00 - 1,50 | SAUDÁVEL | Equilíbrio entre ativos e passivos |
| > 1,50 | EXCELENTE | Folga financeira de longo prazo |

**Valor de referência PMEs**: [AJUSTAR] 1,10 a 1,30

### Liquidez Imediata (LI)

```
LI = Disponibilidades / Passivo Circulante
```

| Faixa | Classificação | Interpretação |
|-------|---------------|---------------|
| < 0,10 | CRÍTICO | Caixa muito baixo |
| 0,10 - 0,20 | ATENÇÃO | Caixa apertado |
| 0,20 - 0,40 | SAUDÁVEL | Caixa adequado |
| > 0,40 | ALTO | Avaliar se há excesso de caixa parado |

**Valor de referência PMEs**: [AJUSTAR] 0,20 a 0,30

**Alertas:**
- < 15%: CRÍTICO - Margem insustentável
- 15% - 25%: ATENÇÃO - Margem baixa
- > 25%: OK para maioria dos setores

### Margem Operacional (EBIT)

```
Margem Operacional = EBIT / Receita Líquida × 100
```

| Faixa | Classificação | Interpretação |
|-------|---------------|---------------|
| < 5% | CRÍTICO | Operação não cobre despesas |
| 5% - 10% | ATENÇÃO | Margem apertada |
| 10% - 20% | SAUDÁVEL | Boa eficiência operacional |
| > 20% | EXCELENTE | Alta eficiência |

**Valor de referência PMEs**: [AJUSTAR] 8% a 15%

### Margem Líquida

```
Margem Líquida = Lucro Líquido / Receita Líquida × 100
```

| Setor | Faixa Típica | Observação |
|-------|--------------|------------|
| Varejo | 2% - 8% | [AJUSTAR] |
| Serviços | 10% - 25% | [AJUSTAR] |
| Indústria | 5% - 15% | [AJUSTAR] |
| Tecnologia | 15% - 35% | [AJUSTAR] |
| [PREENCHER] | [%] | [OBS] |

**Alertas:**
- Negativa: CRÍTICO - Prejuízo
- 0% - 3%: ATENÇÃO - Lucro muito baixo
- 3% - 10%: OK para maioria
- > 10%: BOM

### ROE - Retorno sobre Patrimônio Líquido

```
ROE = Lucro Líquido / Patrimônio Líquido × 100
```

| Faixa | Classificação | Interpretação |
|-------|---------------|---------------|
| < 5% | BAIXO | Retorno menor que renda fixa |
| 5% - 15% | MODERADO | Retorno aceitável |
| 15% - 25% | BOM | Bom retorno para acionistas |
| > 25% | EXCELENTE | Alto retorno (avaliar sustentabilidade) |

**Valor de referência**: Deve ser > Taxa Selic + 5%
**Referência PMEs**: [AJUSTAR] 15% a 20%

### ROA - Retorno sobre Ativos

```
ROA = Lucro Líquido / Ativo Total × 100
```

| Faixa | Classificação | Interpretação |
|-------|---------------|---------------|
| < 3% | BAIXO | Ativos sendo mal utilizados |
| 3% - 8% | MODERADO | Uso aceitável de ativos |
| 8% - 15% | BOM | Boa eficiência de ativos |
| > 15% | EXCELENTE | Alta eficiência |

**Valor de referência PMEs**: [AJUSTAR] 6% a 10%

### EBITDA

```
EBITDA = Lucro Operacional + Depreciação + Amortização
Margem EBITDA = EBITDA / Receita Líquida × 100
```

## 3. Indicadores de Endividamento

Medem a estrutura de capital e capacidade de pagamento de longo prazo.

### Endividamento Geral

```
Endividamento Geral = (Passivo Circulante + Passivo Não Circulante) / Ativo Total × 100
```

| Faixa | Classificação | Interpretação |
|-------|---------------|---------------|
| < 30% | CONSERVADOR | Baixo endividamento |
| 30% - 50% | MODERADO | Endividamento equilibrado |
| 50% - 70% | ALTO | Monitorar capacidade de pagamento |
| > 70% | CRÍTICO | Alto risco de insolvência |

**Valor de referência PMEs**: [AJUSTAR] 40% a 60%

### Composição do Endividamento

```
Composição = Passivo Circulante / (Passivo Circulante + Passivo Não Circulante) × 100
```

| Faixa | Classificação | Interpretação |
|-------|---------------|---------------|
| > 80% | CRÍTICO | Dívidas concentradas no curto prazo |
| 60% - 80% | ATENÇÃO | Muita pressão de curto prazo |
| 40% - 60% | SAUDÁVEL | Boa distribuição de prazos |
| < 40% | EXCELENTE | Dívidas bem alongadas |

**Valor de referência PMEs**: [AJUSTAR] < 60%

### Índice de Cobertura de Juros

```
ICJ = EBIT / Despesas Financeiras
```

| Faixa | Classificação | Interpretação |
|-------|---------------|---------------|
| < 1,0 | CRÍTICO | Não cobre os juros |
| 1,0 - 2,0 | ATENÇÃO | Cobertura apertada |
| 2,0 - 4,0 | SAUDÁVEL | Boa capacidade de pagamento |
| > 4,0 | EXCELENTE | Alta folga financeira |

**Valor de referência PMEs**: [AJUSTAR] > 2,5

### Dívida Líquida / EBITDA

```
Dívida Líquida = Dívida Bruta - Disponibilidades
Índice = Dívida Líquida / EBITDA
```

| Faixa | Classificação | Interpretação |
|-------|---------------|---------------|
| < 1,0x | EXCELENTE | Dívida muito baixa |
| 1,0x - 2,0x | SAUDÁVEL | Endividamento controlado |
| 2,0x - 3,0x | ATENÇÃO | Endividamento elevado |
| > 3,0x | CRÍTICO | Alto risco de dificuldade financeira |

**Valor de referência PMEs**: [AJUSTAR] < 2,0x

## 4. Ciclo Financeiro e Operacional

### Ciclo Operacional

```
Ciclo Operacional = PME + PMR
```

Onde:
- **PME** (Prazo Médio de Estoques) = (Estoque Médio / CMV) × 360
- **PMR** (Prazo Médio de Recebimento) = (Clientes Médio / Receita) × 360

### Ciclo Financeiro (Ciclo de Caixa)

```
Ciclo Financeiro = Ciclo Operacional - PMP
```

Onde:
- **PMP** (Prazo Médio de Pagamento) = (Fornecedores Médio / Compras) × 360

### Valores de Referência por Setor

| Setor | PME (dias) | PMR (dias) | PMP (dias) | Ciclo Financeiro |
|-------|------------|------------|------------|------------------|
| Varejo | 30-60 | 15-30 | 30-45 | 15-45 dias |
| Serviços | 0-15 | 30-60 | 20-40 | 10-35 dias |
| Indústria | 60-120 | 45-90 | 45-60 | 60-150 dias |
