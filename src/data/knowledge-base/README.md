# Knowledge Base - BMV Consultoria

Base de conhecimento para alimentar a IA do sistema BMV Software.

## Estrutura

```
knowledge-base/
├── README.md                    # Este arquivo
├── tributario.md                # Regimes tributários e impostos
├── indicadores-financeiros.md   # KPIs financeiros e valores de referência
├── metodologia-bmv.md           # Metodologia de consultoria BMV
├── alertas-ia.md                # Regras para geração de alertas
└── setores.md                   # Particularidades por setor de atuação
```

## Como usar

Esta base de conhecimento é utilizada pela IA para:

1. **Gerar insights contextualizados** - Com base nos indicadores e regras definidas
2. **Emitir alertas inteligentes** - Seguindo as regras de cada tipo de alerta
3. **Contextualizar análises por setor** - Aplicando particularidades de cada segmento
4. **Orientar diagnósticos** - Usando a metodologia BMV

## Convenções

- Campos marcados com `[PREENCHER]` devem ser completados pela equipe BMV
- Campos marcados com `[AJUSTAR]` podem ser customizados conforme necessidade
- Valores de referência são baseados em PMEs brasileiras
- Atualizar periodicamente conforme mudanças na legislação

## Manutenção

| Arquivo | Responsável | Frequência de Atualização |
|---------|-------------|---------------------------|
| tributario.md | Equipe Contábil | Anual / quando houver mudança |
| indicadores-financeiros.md | Equipe Financeira | Semestral |
| metodologia-bmv.md | Direção BMV | Conforme evolução |
| alertas-ia.md | Equipe de Produto | Conforme feedback |
| setores.md | Consultores | Conforme novos setores |
