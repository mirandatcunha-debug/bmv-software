# Particularidades por Setor

Base de conhecimento sobre características específicas de cada setor de atuação.

## 1. Varejo

### 1.1 Características Gerais

| Aspecto | Descrição |
|---------|-----------|
| Ciclo de vendas | Curto (venda direta ao consumidor) |
| Margens típicas | Menores (compensadas por volume) |
| Estoque | Alta rotatividade necessária |
| Sazonalidade | Alta (datas comemorativas) |
| Capital de giro | Moderado a alto |

### 1.2 Indicadores Específicos

| Indicador | Valor de Referência | Observação |
|-----------|---------------------|------------|
| Margem Bruta | 25% - 45% | Varia por tipo de produto |
| Margem Líquida | 2% - 8% | Margens apertadas |
| Giro de Estoque | 6x - 12x/ano | Depende do produto |
| PME (Prazo Médio Estoque) | 30 - 60 dias | Ideal < 45 dias |
| PMR (Prazo Médio Recebimento) | 15 - 30 dias | Muito cartão de crédito |
| PMP (Prazo Médio Pagamento) | 30 - 45 dias | Negociar com fornecedores |
| Ciclo Financeiro | 15 - 45 dias | Ideal próximo a zero |
| Inadimplência | < 5% | Crediário próprio |
| [PREENCHER] | [Valor] | [OBS] |

### 1.3 Particularidades Tributárias

| Item | Observação |
|------|------------|
| ICMS | Imposto principal, atenção a substituição tributária |
| Crédito de ICMS | Verificar possibilidade de creditamento |
| Simples Nacional | Muitos varejos se enquadram |
| [PREENCHER] | [OBS] |

### 1.4 Riscos Específicos

| Risco | Sinais de Alerta | Ação Recomendada |
|-------|------------------|------------------|
| Estoque parado | Giro < 4x/ano, itens sem venda há 90+ dias | Liquidação, revisão de mix |
| Sazonalidade mal gerida | Falta de caixa pós-alta temporada | Reserva para baixa temporada |
| Dependência de poucos produtos | >40% receita em uma categoria | Diversificação |

## 2. Serviços

### 2.1 Características Gerais

| Aspecto | Descrição |
|---------|-----------|
| Ciclo de vendas | Varia (curto a longo) |
| Margens típicas | Maiores (menor custo de produto) |
| Estoque | Mínimo ou inexistente |
| Sazonalidade | Depende do tipo de serviço |
| Capital de giro | Geralmente menor |

### 2.2 Indicadores Específicos

| Indicador | Valor de Referência | Observação |
|-----------|---------------------|------------|
| Margem Bruta | 40% - 70% | Sem custo de mercadoria |
| Margem Líquida | 10% - 25% | Margens melhores |
| Receita por Colaborador | R$ 15.000 - R$ 50.000/mês | Produtividade |
| Taxa de Ocupação | > 70% | Para serviços com capacidade |
| PMR (Prazo Médio Recebimento) | 30 - 60 dias | Contratos maiores = prazos maiores |
| Churn Rate | < 5%/mês | Para recorrência |
| LTV/CAC | > 3x | Aquisição de clientes |
| [PREENCHER] | [Valor] | [OBS] |

### 2.3 Particularidades Tributárias

| Item | Observação |
|------|------------|
| ISS | Imposto principal (2% a 5% conforme município) |
| Anexo III ou V Simples | Depende do fator R (folha/receita) |
| Lucro Presumido | 32% de presunção |
| [PREENCHER] | [OBS] |

### 2.4 Riscos Específicos

| Risco | Sinais de Alerta | Ação Recomendada |
|-------|------------------|------------------|
| Dependência de pessoas-chave | 1-2 pessoas com todo conhecimento | Documentação, treinamento |
| Concentração de clientes | Top 3 > 60% receita | Diversificação |
| Precificação por hora | Dificuldade de escalar | Migrar para pacotes/valor |
| [PREENCHER] | [Sinais] | [Ação] |
